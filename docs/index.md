# astromiedust

`astromiedust` computes optical efficiency coefficients for circumstellar dust
particles and uses them to estimate equilibrium temperatures, thermal emission,
radiation-pressure beta factors, and blowout sizes.

The preferred public import is:

```python
import astromiedust as amd
```

The old `astrodust_optprops` package name is kept only as a compatibility layer
for older code.
