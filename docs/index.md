# astromiedust

`astromiedust` computes optical efficiency coefficients for circumstellar dust
particles and uses them to estimate equilibrium temperatures, thermal emission,
radiation-pressure beta factors, and blowout sizes.

The preferred public import is:

```python
import astromiedust as opt
```

The old `astrodust_optprops` package name is kept only as a compatibility layer
for older code.

## First calculation

```python
import numpy as np
import astromiedust as opt

star = opt.Star(name="Fomalhaut", lum_suns=16.6, mass_suns=1.92, temp=8500)
matrl = opt.Material(qsil=0.4, qice=1.0, mpor=0.7, emt="maxwell-garnett")

wavs = np.logspace(0, 4, 300)   # wavelength grid, um
diams = np.logspace(0, 4, 41)   # particle diameters, um
dists = np.logspace(0, 3, 43)   # distance from star, au

prtl = opt.Particles(diams=diams, wavs=wavs, dists=dists, matrl=matrl)
prtl.calculate_all(star)

print(prtl.Qabs.shape)
print(prtl.temps.shape)
print(prtl.betas.shape)
```

`Particles.calculate_all(star)` runs the common workflow: scattering
coefficients, equilibrium temperatures, beta factors, blowout diameters, and
blackbody spectral radiance on the requested wavelength grid.
